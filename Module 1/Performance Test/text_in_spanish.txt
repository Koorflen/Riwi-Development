La manera en que analice como desarrollar este codigo fue: primero crear un inventario el cual seria una lista, en esta lista estarian todos los productos, cada producto seria una diccionario, el diccionario tiene tres llaves las cuales son 'name','price','quantity', las cuales corresponden al nombre el precio y la cantidad de unidades respectivamente, tras esto cree un menu inicial en el cual el usuario decide que hacer

Menu Principal: en este estan las opciones de añadir,consult,actualizar,remover, y calcular valor total. 

Añadir: el usuario al añadir productos por primera vez, tiene que añadir una cantidad de productos mayor a 5, los 5 primeros productos sobreescriben los 5 borradores que estaban la lista, de ahi en adelante se van agregando como diccionarios nuevos, pero si el usuario no va a añadir por primera vez osea que antes ya habia añadido, no sobreescribe los 5 primeros, solo agrega nuevos.

Consult:Como el monto minimo es de 6 productos en el inventario primero se certifica si el inventario tiene los 5 productos que habian como borrador, si es asi entonces se redirige al menu principal y se le dice que no a agregado productos para consultar, si el inventario ya tiene productos, esta funcion le solicita el nombre del producto a buscar, este se busca en el inventario y si se encuentra se muestra la informacion de este (nombre,precio,cantidad), si no se encuentra redirige al menu principal y deja el mensaje de que ese producto no esta en el inventario.

Actualizar:Se verifica si el inventario tiene productos de la misma manera que se hizo en la funcion 'consult', tras confirmar que el inventario tiene productos se le solicita el nombre del producto que desea actualizar, se busca este producto en el inventario si no se encuentra redirige al menu principal y se muestra el mensaje de que el producto no esta en el inventario, si es producto se encuentra se pregunta que dato quiere actualizar(precio/cantidad), tras la eleccion se solicita el nuevo dato y luego se muestra la informacion del producto mostrando su dato anterior y el dato ya actualizado.

Remover:se verifica la cantidad de productos en el inventario, si hay 5 se redirige al meni principal y se le dice que no ha añadido productos, si tiene 6 se le redirige al menu principal y se le dice que no se pueden tener menos de 6 productos. Si al inventario se le pueden remover productos entonces se le pedira en nombre del producto que desea remover, este se buscara en el inventario si no se encuentra se le redirige al menu principal y se le dice que este producto no se encuentra en el inventario, si el producto se encuentra se muestra la informacion del producto y se solicita confirmacion para removerlo, si la confirmacion es afirmativa el producto es removido del inventario y se muestra un mensaje indicando que el proceso fue exitoso, si la confirmacion es negativa se le redirige al menu principal

Calcular valor total:se confirma si el inventario tiene productos, si no tiene productos se le redirige al menu principal y se le indica que no tiene productos ingresados aun, si el inventario tiene productos se le pregunta que desea calcular, si el valor total de un producto o el valor total del inventario. si se desea consultar el valor total de un producto se pregunta por que producto desea que se le haga el calculo, se busca si esta en el inventario, si no esta se le redirige al menu principal y se le dice que este producto no esta en el inventario, si se encuentra se le muestra los datos de este producto y el calculo del valor total. si desea consultar el calculo del valor total del inventario, se comienza a recorrer todo el inventario y se va guardando en una varibale el valor total de cada producto tras recorrer toda la lista, se imprime un mensaje con el valor total del inventario
